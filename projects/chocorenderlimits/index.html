Projects / Chocorenderlimits (SDL2)

Chocorenderlimits is a fork of Chocolate Doom from RestlessRodent (formerly GhostlyDeath), created for testing static limits of vanilla Doom engine. It have implemented counters of visplanes, drawsegs and other information, needed for checking limit values.

This version has been adapted to SDL2 library, which gives painless full screen toggling, as well as few other benefits.

Download

For Windows (both 32-bit and 64-bit editions): chocorenderlimits-sdl2-win32.zip
Source code (only Windows / Code::Blocks IDE is supported): chocorenderlimits-sdl2-src.zip
GitHub repository: JNechaevsky/ChocoRenderLimits

In-game menu with counters can be activated by pressing ~ key.
