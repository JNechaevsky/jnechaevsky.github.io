Projects / Prussian blue palette for Doom

Slightly different and more cyanish blue colors for Doom. This colors were appearing in various total conversions along with slightly different red colors, but in this PWAD only blue colors has been changed.

I honestly have no idea who created this color set initially, but hopefully it will never be lost again!

Download

Archive with palette: bluepal.zip

To use it, just load the game with -file command line parameter like:
doom.exe -file bluepal.wad

Or put into suitable "autoload" folder.
