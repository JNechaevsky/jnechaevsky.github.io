Projects / Prussian blue palette for Doom

About

Slightly different and more cyanish blue colors for Doom. These colors appeared in various total conversions along with slightly different red tones, but in this PWAD only blue colors have been changed.

I honestly have no idea who initially created this color set, but hopefully, it will never be lost again!

Download

Archive with palette: bluepal.zip

To use it, just load the game with the -file command-line parameter like:
doom.exe -file bluepal.wad

Or put the file into a suitable "autoload" folder.
