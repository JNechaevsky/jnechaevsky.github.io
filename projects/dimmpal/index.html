Projects / Optimized palettes for Doom, Heretic, Hexen and Strife

About

Optimized game palettes for bright modern LCD monitors. Original colors are preserved, while saturation and contrast are improved.

It is recommended to use with software rendering source ports like vanilla Doom for DOS, Chocolate Doom, Crispy Doom, ZDoom and PrBoom+.

Download

Archive with all palettes: dimmpal.zip

To use them, just load the game with -file command line parameter for:

Doom:doom-pal.wad
Heretic:her-pal.wad
Hexen:hex-pal.wad
Strife:strf-pal.wad
