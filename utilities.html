A collection of MIT-licensed utility tools written by Polina "Aura" N. and Julia Nechaevskaya.

Allman Formatter
Automatically reformats source code into the Allman style with 4‑space indentation.

Bit Shift Calculator
Shift decimal, hexadecimal or binary integer values left and right.

Countdown Timer
Set a countdown, start or pause it, and adjust the time with presets or manual inputs.

Deposit Calculator
A simple deposit calculator; for estimating ruble-denominated returns based on principal amount, interest rate, and term.

DOCX Unprotector
Remove editing restrictions from .docx files (including password protection).

Hex / Dec Converter
Convert hexadecimal values to decimal and decimal values back to HEX.

Image Compressor
Reduce PNG and JPG file size.

Non‑Cyrillic Checker
Highlight and fix Latin letters in Russian full‑name lists to ensure clean Cyrillic-only formatting.

Password Generator
Generate password variants with configurable length, character sets, and optional unique characters.

PDF Splitter & Merger
Split a PDF into JPG pages (ZIP) or merge PNG/JPG images into a single PDF.
